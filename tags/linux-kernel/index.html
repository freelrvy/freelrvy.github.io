linux kernel配置

自從重裝系統之後總是有着若有若無的卡噸，很煩人，所以這回是決定單挑gentoo了，

當然是先要再虛擬機裏安裝了
照着handbook硬啃真是台痛苦了（中文手冊已經不維護了，所以還是看得英文的啊啊啊啊啊），其餘都還好，就是kernel配置有些鬱悶，正好另一個基佬@fyyz_me 也在裝gentoo，所以跟他溝通了一下，他對與內核配置方面也很頭疼，不過倒是暫時永樂一種比較穩妥的方法->>把Arch的kernel-config文件先cp過來用一下，等深入瞭解之後再定製，
cat /proc/config.gz | gzip -d > /home/config /* 這裏的config就是kernel的配置文件了 /*
之後再鄙夷就暫時Ok了
